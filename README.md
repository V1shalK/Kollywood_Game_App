## Features

- Homepage with username and score display  
- Mode selection page with single-player and multiplayer options  
- Single-player mode:
  - Displays the first letters of hero, heroine, movie, and song
  - Data fetched from the database
  - 25 points per correct answer (100 points per round)
  - 9 lives (based on the letters in "KOLLYWOOD")
  - 10-minute timer
- Multiplayer mode:
  - One user provides the data for another
  - No database fetch in this mode
- Database: PostgreSQL hosted on Neon
- Separate frontend and backend folders within a single repository

---

## Tech Stack

**Frontend:** HTML, CSS, JavaScript  
**Backend:** Node.js, Express.js  
**Database:** PostgreSQL (Neon)
